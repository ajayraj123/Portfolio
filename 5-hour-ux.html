Framework that helps designers in tackling daily design problems

Category

Project

Overview

I created this framework to tackle daily changes I faced as a designer. Not everyday in the life of a designer is ideal. UX is a multidisciplinary field. Type of work designers do and the pace at which they are doing changes based on the requirements. This framework addresses those challenges.

It is based on timeboxing and many other principles. It helps in solving common issues designers face such as Analysis Paralysis, Overthinking Trap, Uncertainty, Einstellung Effect etc.

There are 2 versions of the framework. Initially it was 4 Hours and the new improvised version is 5 Hours. Read in detail about the frameworks below.
